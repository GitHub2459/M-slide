Sistema de Probabilidade - Implementação

A função generateProbabilityQuestion é responsável por criar questões de probabilidade:

function generateProbabilityQuestion(min, max, level) { // 7 tipos de questões pré-definidas // Seleção baseada na dificuldade // Geração de alternativas diversificadas // Retorno da questão formatada }

Tipos de Questões:

  • Moeda: Probabilidade de cara/coroa
  • Dado par/ímpar: Probabilidade de números pares/ímpares
  • Dado específico: Probabilidade de um número específico
  • Cartas: Probabilidade de tirar um Ás
  • Naipes: Probabilidade de tirar uma carta de naipe específico

Fórmulas de Probabilidade

Fórmula Base:

P(evento) = número de casos favoráveis / número de casos totais

Exemplos Práticos:

  • Moeda: P(cara) = 1/2
  • Dado: P(número par) = 3/6 = 1/2
  • Baralho: P(Ás) = 4/52 = 1/13
  • Naipes: P(Copas) = 13/52 = 1/4

O sistema simplifica as frações automaticamente para apresentar as respostas na forma mais reduzida possível.

Geração de Alternativas

Métodos Utilizados:

  • Frações equivalentes: Simplificação da resposta correta
  • Denominadores variados: Geração de frações com diferentes denominadores
  • Valores próximos: Frações com valores próximos ao correto

Controle de Qualidade:

  • Garante alternativas distintas
  • Evita duplicatas
  • Mantém consistência com o nível de dificuldade
// Adicionar alternativas incorretas while (alternatives.size < 4) { let numAlt, denAlt; if (Math.random() < 0.5) { // Frações com denominadores diferentes denAlt = rnd(2, 12); numAlt = rnd(1, denAlt - 1); } else { // Frações com valores próximos denAlt = simplified.den + rnd(-3, 3); if (denAlt < 2) denAlt = 2; numAlt = Math.max(1, Math.min(denAlt - 1, Math.round((simplified.num / simplified.den) * denAlt))); } // ... adiciona alternativa se for diferente da correta }

Sistema de Dificuldade

Distribuição por Nível:

  • Fácil: Questões simples (moeda/dados)
  • Médio: Questões intermediárias (mais tipos de dados)
  • Difícil: Todas as categorias (incluindo baralho)

Adaptação Automática:

  • Intervalos numéricos ajustáveis
  • Complexidade das questões escalável
  • Alternativas proporcionalmente desafiadoras
// Selecionar tipo de questão baseado no nível let typeIndex; if (level === 'facil') { typeIndex = rnd(0, 4); // Moeda ou dados } else if (level === 'medio') { typeIndex = rnd(0, 5); // Inclui mais tipos de dados } else { typeIndex = rnd(0, questionTypes.length - 1); // Todos os tipos }
